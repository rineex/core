# Auth Core

Core authentication and authorization package for Rineex framework.

## Features

- Authentication abstractions
- Authorization utilities
- Social login support
- OTP functionality
- Passwordless authentication
- SSO integration

## Installation

```bash
pnpm add @rineex/auth-core
```

## Development

```bash
# Build the package
pnpm build

# Run tests
pnpm test

# Lint code
pnpm lint

# Check types
pnpm check-types
```
